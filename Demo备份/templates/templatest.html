控制语句

{% if age > 18 %} 已经成年 {% elif age == 18 %} 正好成年 {% else %} 未成年 {% endif %}

遍历字典

forloop是django模块提供的一个变量，长和if一起用

forloop

{% for one in hobby %}

{{forloop.counter}}.{{one}}

{% endfor %}

forloop if for 一起用

{% for one in hobby %} {% if forloop.first %}

{{ one }}

{% elif forloop.last %}

{{ one }}

{% endif %} {% endfor %}
